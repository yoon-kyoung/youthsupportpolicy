## 세션 3 — AI 챗봇 랜딩 페이지 그라디언트 배경

### 변경 파일
- `src/chatbot/ChatBotView.jsx` — 랜딩 섹션 배경 및 레이아웃
- `src/App.jsx` — 비데스크탑 레이아웃 높이 수정

### 구현 내용

#### 반응형 그라디언트 배경 (cqw 기반)
- 파랑(`#4AA8FF`)·초록(`#19CEBD`) 두 개의 원형 orb div를 `position:absolute`로 배치
- `width:28.8cqw / 19.4cqw`, `aspectRatio:1`, `filter:blur(12cqw / 8cqw)` — 컨테이너 너비 비례 반응형
- `containerType:'inline-size'` 부모 → cqw 쿼리 기준 제공
- `overflow:'hidden'` — blur 번짐을 컨테이너 경계에서 클리핑, 스크롤 없음

#### orb 위치 (Figma 원본 → 최종)
| | left | top |
|---|---|---|
| 파랑 orb | 24.9% | 52% (원본 61.2%) |
| 초록 orb | 43.2% | 46% (원본 55.7%) |

상단 여백을 더 넓게 하고 검색창이 잘 보이도록 약 9pp 상향 조정.

#### 레이아웃 버그 수정 (핵심)
- **원인**: 비데스크탑 외곽 div가 `minHeight:"100vh"` → flex 자식에게 확정 높이를 부여하지 않음
  → ChatBotView `height:'100%'`가 content 높이(auto)로 해석 → 랜딩 페이지가 content 만큼만 그려지고 아래에 빈 회색 영역 노출
- **수정**: `minHeight:"100vh"` + `overflow:"hidden"` → `height:"100vh"` + `overflow:"hidden"`
  → flex chain에 확정 높이 전달 → ChatBotView가 뷰포트 전체를 꽉 채움

#### 콘텐츠 중앙 정렬
- 이전: `minHeight:'100%'`, 세로 정렬 없음 → 내용이 상단에 붙음
- 변경: `height:'100%'`, `justifyContent:'center'`, `alignItems:'center'`, `padding:'0 24px'`
  → 어떤 화면 비율·환경에서도 가로세로 중앙 배치

#### 기타
- 면책 문구 색상: `#d1d5db` → `#94A3B8` (서브텍스트 색상, 가이드 일치)
