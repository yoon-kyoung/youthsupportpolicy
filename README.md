# 청년지원정책 안내 웹사이트

우리 팀이 만든 청년지원정책 안내 웹사이트입니다.

---

## 이 브랜치(feature/chatbot-admin)에서 추가한 것

> **종권이가 작업한 내용입니다. 팀원분들 확인 후 메인에 합칠 예정이에요!**

### 우리가 한 것

| 기능 | 설명 |
|------|------|
| **AI 챗봇** | "27살 서울인데 월세 지원 있어?" 같이 자연어로 물어보면 AI가 맞춤 정책을 찾아줘요 |
| **스트리밍 응답** | ChatGPT처럼 글자가 실시간으로 나와요 (답변 기다리는 동안 안 답답함) |
| **LLM 정책 선별** | AI가 직접 "이 사람한테 이 정책이 맞겠다" 판단해서 골라줘요 |
| **모델 선택** | 무료 AI 모델 자동 선택 (비용 0원) + 수동 변경 가능 |
| **버튼 폴백** | AI 서버 장애 시 → 나이/지역/분야 버튼으로 자동 전환 (항상 작동) |
| **관리자 페이지** | 사용량, 토큰, 비용 확인 + 일일 한도 자동 차단 |
| **정책 카드** | 추천된 정책을 카드 형태로 보기 좋게 표시 |

### 어떻게 작동하나요?

```
사용자가 질문 입력
    ↓
우리 서버(Vercel)가 AI한테 물어봄
    ↓
AI가 2,600개 정책 중에서 맞는 거 골라줌
    ↓
답변 + 정책 카드를 화면에 표시
```

- 모든 API 키는 서버에만 있어서 **보안 걱정 없음**
- 무료 모델 사용이라 **비용 0원**
- 하루 150회, 분당 20회 제한 있음 (남용 방지)

## 팀원분들이 해주셔야 할 것

### 1. 위 링크에서 테스트해보기
- 챗봇에 아무 질문이나 해보세요 (예: "25살 경기도 취업 준비 중")
- 이상한 점 있으면 종권이한테 말해주세요

### 2. 동의해주기
- 확인 다 했으면 **"괜찮다, 합치자"** 한마디 해주시면 됩니다
- GitHub에서 댓글 달아도 되고, 카톡으로 말해도 됩니다

### 3. 합치기 (merge)
- 팀원 모두 동의하면 이 브랜치를 **main(메인)**에 합칩니다
- 합친다 = 우리 웹사이트 최종 버전에 챗봇이 추가된다는 뜻이에요
- 합치는 건 종권이가 할게요, 동의만 해주시면 됩니다!

## 로컬에서 실행하고 싶을 때 (개발자용)

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 열면 됩니다.

## 사용 기술

- React + Vite (웹사이트)
- Vercel (배포)
- OpenRouter API (AI 챗봇 - 무료 모델)
